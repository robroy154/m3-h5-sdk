Interface IWidgetSettings<T>

Represents the widget settings.

Note that all settings values must be possible to serialize to JSON.

isSettingsEnabled

  • isSettingsEnabled(): boolean

  • Gets a value that indicates if settings are currently enabled. The value depends on the following factors: if settings are enabled in the definition, if settings are enabled for a published widet, if the widget is in a publish / edit published state in the client. This means that the value might change over time when working with a published widget.

    Returns boolean

    True if settings are enabled.
